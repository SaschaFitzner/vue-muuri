# vue-muuri

Example respository to show Vue.js working together with muuri.
The base was an Vue.js Hello World PWA.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Lints and fixes files

```
yarn run lint
```
